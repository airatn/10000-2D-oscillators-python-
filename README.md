# 10000-2D-oscillators-python-
Modeling wave propagation in a 2D elastic environment

Естественным шагом является переход от одномерной к двухмерной модели, то есть к упругой системе, распространение волн в которой происходит на плоскости. С физической точки зрения, в ее основе лежит структура приведенная на рисунке:

<img src="https://drive.google.com/uc?export=view&id=1V4N2aD6LOos3hWvFA3T09kTQWGxnWALb" width="900">

параллельно ей, аналогичными пружинными связями присоединяется такая же цепь связанных тел, и так много раз. В итоге, с мы получаем упругую плоскость показанную на рисунке:

<img src="https://drive.google.com/uc?export=view&id=1pkndAN6s3Z2AoMXvytAQOHpFWUoJ8GEE" width="700">

при приближении мы видим следующую структуру:

<img src="https://drive.google.com/uc?export=view&id=1bpMDMahV-yj_xUOaG2H7nwsEBmPWRGs7" width="600">

Данная модель позволяет воспроизводить гораздо более сложные волновые процесс, а с визуальной точки зрения существенно более наглядные.
